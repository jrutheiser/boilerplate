*** Elements ***

Elements are used for small re-usable styles. Almost any page can be broken
down into smaller elements.

To name a few:
* Buttons
* Tabs
* Labels
* Badges
* Alerts
* Modals
* Select boxes
* Tooltips
* Navigation
* Sidebar
* Inputs
* Range sliders
* Radio buttons
* Checkboxes
* Tables
* Sliders / Carousels
* Lightboxes
* Dropdowns
* Progress bars
